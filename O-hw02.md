# 운영체제 복합문제 풀이

---

## 문제 (1)

3개의 프로그램이 모두 메모리에 적재되어 있다고 할 때, 다중프로그래밍 운영체제에서 3개의 프로그램 A, B, C가 실행되는 과정을 그려라. 참고로 프로그램 A, B, C 순서로 처리하되, 프로그램의 실행 중 입출력이 발생하면 다른 프로그램을 실행시킨다. 중단된 프로그램의 입출력 작업에 완료되었을 경우 현재 실행중인 프로그램을 중단시키고, 입출력으로 중단된 프로그램을 다시 실행시키도록 하라.

---

## 풀이

### 실행 규칙

- **A → B → C 순서로 시작**
- **실행 중 I/O 발생 시 다른 프로그램 실행**
- **I/O 완료 시 즉시 CPU 선점**

---

### 실행 흐름

```
시간 →

A 실행 → A I/O 발생  
→ B 실행 → B I/O 발생  
→ C 실행 → C I/O 발생  

→ A (I/O 완료) → 실행 → 종료  
→ B (I/O 완료) → 실행 → 종료  
→ C (I/O 완료) → 실행 → 종료
```

---

### CPU 실행 타임라인

```
| A | A | A | A | B | B | B | C | C | C | A | A | B | B | C | C |
```

---

### 프로그램별 상태 흐름

```
A → [CPU][CPU][CPU][CPU][I/O][I/O][I/O][I/O][I/O][I/O][CPU][CPU] → 종료
B →         [CPU][CPU][CPU][I/O][I/O][I/O][I/O][I/O][I/O][CPU][CPU] → 종료
C →                 [CPU][CPU][CPU][I/O][I/O][I/O][CPU][CPU][CPU] → 종료
```

---

### 해석

- A 실행 중 I/O 발생 → B 실행  
- B 실행 중 I/O 발생 → C 실행  
- C 실행 중 I/O 발생 → A의 I/O 완료 → A 선점 실행  
- 이후 동일한 방식 반복  

➡️ CPU는 계속 다른 프로그램을 실행하므로 **유휴 시간이 발생하지 않는다.**

---

### 답

다중 프로그래밍 운영체제에서는  
I/O가 발생할 때마다 다른 프로그램으로 전환하고,  
I/O가 완료된 프로그램이 즉시 CPU를 선점하여 실행이 이어진다.  

따라서 CPU가 지속적으로 활용되는 실행 구조가 형성된다.

---

## 문제 (2)

실행 결과에 따라 아래 2개의 표를 채워라.  
표에서 처리율은 단위 시간당 처리한 프로그램의 개수이고,  
응답 시간이란 프로그램이 준비상태에서 실행이 완료될 때까지의 시간이며,  
평균 응답 시간이란 3개 프로그램의 응답 시간에 대한 평균값이다.

---

## 풀이

### 1) 배치 운영체제

- **총 실행 시간:** 30  
- **CPU 유휴 시간:** 14  

CPU 이용률 계산:  
(30 - 14) / 30 = 16 / 30 ≈ **53%**

처리율:  
3 / 30 = **0.1 (개/시간)**

응답 시간:  
- A: 11  
- B: 22  
- C: 30  

평균 응답 시간:  
(11 + 22 + 30) / 3 = **21**

---

### 2) 다중 프로그래밍 운영체제

- **총 실행 시간:** 16  
- **CPU 유휴 시간:** 0  

CPU 이용률:  
16 / 16 = **100%**

처리율:  
3 / 16 ≈ **0.1875 (개/시간)**

응답 시간:  
- A: 11  
- B: 13  
- C: 16  

평균 응답 시간:  
(11 + 13 + 16) / 3 = **40 / 3 ≈ 13.3**

---
